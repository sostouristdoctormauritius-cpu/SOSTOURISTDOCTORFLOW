# Assets Directory

This directory should contain all the assets used in the application:

- Images
- Icons
- Fonts
- Animations

## Structure

```
assets/
├── images/
│   ├── icons/
│   ├── logos/
│   └── illustrations/
├── fonts/
└── animations/
```

## Usage

Import assets directly in components:

```javascript
import logo from '../assets/images/logos/logo.png';
```
